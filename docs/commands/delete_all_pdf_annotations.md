doc_body:
Delete all of the built-in PDF annotations (this is different from sioyek annotations, here we mean the PDF annotations created in other PDF viewers or using sioyek's embed_annotations). The @command(delete_intersecting_pdf_annotations) command can be used to delete only the annotations that intersect with the selected rectangle.
